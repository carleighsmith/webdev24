Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

It is toggles between relative and fixed, depending on the scroll position. it positioned relative until a given offset position is met in the viewport - then it fixed in place.

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

Relative to its normal position. we can apply properties like top, down , left, right. These four(TDLR) properties causes it to be adjusted away from its original position.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

It positioned relative to the VIEWPORT,( means it always stays in the same place during the scrolling). Four properties are used to position the element. this does not leave a gap in the page.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

It positioned relative to the nearest positioned ancestor(not same as fixed).But, if it has no positioned ancestors, it moves along with page scrolling.
