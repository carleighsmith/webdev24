Web Security Information

Welcome to our web security information page. Here, we'll discuss important practices for using the web safely and highlight some of the information that web applications can access about your computer.

Information Accessed by Web Apps:

  • JavaScript can access the browser user agent using the navigator.userAgent property.
  • JavaScript can access the screen's pixel Depth using the screen.pixelDepth property.
  • JavaScript can access the language using the navigator.language property.
  • JavaScript can access the app Version using the navigator.appVersion property.
  • JavaScript can access the avail Width using the screen.availWidth property.
  • JavaScript can access a user's geolocation using the navigator.geolocation property.
There is a breadth of information on a user's computer that a web app can easily access.

Practices for Using the Web Safely:

  1. Federal Trade Commission: Protect Your Personal Information and Data - The website of the Federal Trade Commission offers thorough instructions on how to secure your data and personal information, along with helpful hints and resources to prevent identity theft, data breaches, and other online dangers.
  2. StaySafeOnline: Online Safety Basics - The Online Safety Basics resource from StaySafeOnline covers fundamental subjects like making strong passwords, spotting phishing attempts, and safeguarding personal information. It provides users of all levels with crucial advice and best practices to improve their online safety and security.
  3. US-CERT: Cybersecurity Tips -The Cybersecurity Best Practices website from CISA offers resources and authoritative advice on risk management, incident response, and critical infrastructure security to assist people and organizations in putting effective cybersecurity measures in place.
